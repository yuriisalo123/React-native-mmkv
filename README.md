# react-native-mmkv

An efficient, small mobile key-value storage framework developed by WeChat. Works on Android and iOS.

## Installation

```sh
npm install react-native-mmkv
```

## Usage

```js
import Mmkv from "react-native-mmkv";

// ...

const result = await Mmkv.multiply(3, 7);
```
